# Rock, Paper and Scissors - Frontend

Rock, paper and scissors frontend

## Setup for development

```sh
yarn install
yarn run start
# Open your browser in: http://localhost:4200
```

## Build for production

```sh
# Modify your environments/environment.prod.ts
yarn run build --prod
# Publish files generated on: dist/ folder
```
